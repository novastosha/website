Manual & Rules

                

                    
OBJECTIVE: Compress the data until the Current Cost is lower than the Target.


                    

                        THE COST MODEL

                    
Every item on the main tape costs 1
                            unit. Your goal is to replace many items with fewer tokens.


                    
    
                        
  • 
                            [RLE] Run-Length Encoding
    
                            Compresses sequential identical characters.
    
                            
                                • Cost: Always 2 units (1 for count, 1 for char).
    
                                • Example: "AAAAA" (5 cost) → "5A" (2 cost).
                                
                        
    • 

                        
  • 
                            [PATTERN] Dictionary Definition ($)
    
                            Saves a sequence to memory to be used multiple times.
    
                            
                                • Storage Cost: You pay the cost of the sequence once to
                                define it.
    
                                • Ref Cost: Placing the reference (e.g., $1) costs only 1
                                    unit.
    
                                • Automatic: Defining a pattern automatically replaces ALL instances of
                                it on the tape.
                                
                        
    • 
                    


                    

                        OPTIMIZATION STRATEGY

                    

                        Standard: Define "AAAAAB" as $1.

                        Cost: 6 (Storage) + 1 (Ref) = 7.
                    

                    

                        Optimized: Compress "AAAAA" → "5A" first.

                        Then define "5AB" as $1.

                        Cost: 3 (Storage) + 1 (Ref) = 4.

                        Tip: Compress the inside of your patterns before defining them!
                    

                    

                        Input Controls:

                        • Double-Click or Double-Tap: Smart
                            Select —
                        automatically selects the largest continuous sequence of identical symbols.

                        • Click & Drag: Select a custom range
                        (Desktop).

                        • Tap Start & End: Select a range
                        (Mobile/Touch).
                    

                

                
            

        



        

            
Compression Puzzle

            

                Compression Puzzle is a logic game about structure.
                

                You are given a fixed target string. Your goal is to recreate it exactly using the smallest possible
                description — by grouping patterns, repeating runs, and defining
                    reusable pieces.
                

                

                Can you describe the same data using less information?
